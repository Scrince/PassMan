Scope
-----

The macOS release packages the Python/PyQt6 application as a .app bundle
inside a compressed DMG. The vault format is unchanged. Frozen builds store
the vault at:

  ~/Library/Application Support/PassMan/vault.dat

with automatic backup:

  vault.dat.bak

Code Signing vs Release Signing
-------------------------------

  - OpenPGP (.asc) signatures prove the DMG bytes match what the release key
    holder published. Verify them before install.
  - Apple code signing is separate. These DMGs are ad-hoc codesigned for local
    launch on Apple Silicon; they are not Developer ID signed or notarized
    unless separately documented. Gatekeeper may still warn on first open.
  - After building a fresh .app on Apple Silicon, ad-hoc sign before packaging:

      codesign --force --deep --sign - dist/macos-arm64/PassMan.app


Operational Notes
-----------------

  - Do not publish vault.dat with application binaries.
  - Prefer the universal DMG when the target Mac architecture is unknown.
  - Prefer the arm64 DMG on Apple Silicon for the smaller download.
  - Verify hashes and OpenPGP signatures before unlocking a real vault.
